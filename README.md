# flutter_webrtc_agora_example

This project is a WebRTC example with AgoraSDK.

## Getting Started

### Install

```
$ git clone https://github.com/yukitaka13-1110/flutter_webrtc_agora_example.git
```

### Build

```
$ flutter run --release
```
